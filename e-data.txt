E-commers Tables
1. Users:
Field	Type
user_id	UUID PK
name	varchar
email	varchar UNIQUE
password_	varchar
phone	varchar
role	enum("user","admin")
created_at	timestamp

2. Addresses:
Field	Type
address_id	UUID PK
user_id	UUID FK
line1	varchar
line2	varchar
city	varchar
state	varchar
pincode	varchar
country	varchar

3. Categories:
Field	Type
category_id	UUID PK
name	varchar

4. Brands
Field	Type
brand_id	UUID PK
name	varchar
5. Products
Field	Type
product_id	UUID PK
category_id	UUID FK
brand_id	UUID FK
name	varchar
description	text
specs	jsonb

6. ProductVariants
Field	Type
variant_id	UUID PK
product_id	UUID FK
name	varchar
sku	varchar UNIQUE
price	int
discount_price	int nullable
stock_status	enum("in_stock","low_stock","out_of_stock")
weight	int nullable

7. Stock
Field	Type
stock_id	UUID PK
variant_id	UUID FK UNIQUE
quantity	int
updated_at	timestamp



8. ProductImages
Field	Type
image_id	UUID PK
product_id	UUID FK
image_url	varchar
position	int

13. Coupons
